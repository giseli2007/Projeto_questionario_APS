# Projeto_questionario_APS
ste sistema é uma aplicação web desenvolvida como parte de uma Iniciação Científica para otimizar a coleta e análise de dados sobre o acolhimento na Atenção Primária à Saúde (APS) em Rondônia. O objetivo é permitir que cidadãos avaliem o atendimento recebido, gerando dados estruturados para análise de políticas públicas.
